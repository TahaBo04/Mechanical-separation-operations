Formulas used

  • K = a·√( g·ρ·(ρp−ρ)·D³ / μ² )
  • Regime by K: <K1 → Stokes (b₁=24,n=1) ; K1–K2 → Interm. (18.5,0.6) ; ≥K2 → Newton (0.44,0)
  • vt = [ 4Dg(ρp−ρ)/(3b₁ρ) ]1/(2−n) · ( μ/(ρD) )n/(2−n)
  • Re = ρ vt D / μ ; Cd = b₁ / Reⁿ ; F = ½ ρ Cd A vt², A = πD²/4
